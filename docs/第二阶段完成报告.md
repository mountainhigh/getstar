## 📊 数据库设计

### 表关系图

```
families (家庭)
  └─> children (孩子)
        ├─> habits (习惯)
        ├─> check_ins (打卡记录)
        └─> user_badges (用户称号)

users (用户)
  └─> families (家庭)

rewards (礼物)
  └─> reward_exchanges (兑换记录)

check_ins (打卡记录)
  └─> coin_records (金币记录)
```

### 新增表结构

#### badges (称号表)
```javascript
{
  _id: String,
  name: String,        // 称号名称
  icon: String,        // 称号图标
  description: String,  // 称号描述
  condition: String,    // 解锁条件
  requiredPoints: Number, // 所需积分
  unlockType: String,  // unlock_count | unlock_streak | unlock_points
  unlockValue: Number  // 解锁值
}
```

#### user_badges (用户称号表)
```javascript
{
  _id: String,
  _openid: String,     // 用户openid
  childId: String,     // 孩子ID
  badgeId: String,     // 称号ID
  unlockTime: Date,    // 解锁时间
  isEquipped: Boolean  // 是否佩戴
}
```

#### rewards (礼物表)
```javascript
{
  _id: String,
  name: String,        // 礼物名称
  icon: String,        // 礼物图标
  category: String,    // 分类
  cost: Number,        // 金币价格
  stock: Number,       // 库存
  description: String  // 描述
}
```

#### reward_exchanges (兑换记录表)
```javascript
{
  _id: String,
  _openid: String,     // 用户openid
  childId: String,     // 孩子ID
  rewardId: String,    // 礼物ID
  cost: Number,        // 兑换时花费金币
  createTime: Date     // 兑换时间
}
```

#### coin_records (金币记录表)
```javascript
{
  _id: String,
  _openid: String,     // 用户openid
  childId: String,     // 孩子ID
  amount: Number,      // 金币变动数量
  type: String,        // check_in | reward_exchange | undo
  relatedId: String,   // 关联ID（打卡记录ID或兑换记录ID）
  createTime: Date     // 创建时间
}
```
