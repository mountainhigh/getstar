## 🔄 工作流程

### 解锁流程
1. **打卡成功** → 调用 `checkBadgeUnlock` 云函数
2. **云函数检查** → 计算总打卡次数和连续天数
3. **判断解锁** → 遍历所有称号,检查解锁条件
4. **添加记录** → 将新解锁的称号添加到 `user_badges` 表
5. **奖励星星** → 每解锁一个称号奖励50星星
6. **显示动画** → 前端显示解锁弹窗动画

### 佩戴流程
1. **用户点击** → 在称号图鉴页面点击未佩戴的称号
2. **更新数据库** → 将称号ID写入 `children.currentBadgeId`
3. **同步显示** → 首页实时显示当前佩戴的称号

## 📖 使用说明

### 查看称号图鉴
1. 首页点击星星卡片中的称号
2. 或访问: `pages/badge-collection/badge-collection`

### 佩戴称号
1. 进入称号图鉴页面
2. 点击已解锁的称号
3. 点击"佩戴称号"按钮

### 解锁称号
1. 完成习惯打卡
2. 系统自动检测是否满足解锁条件
3. 如满足,自动显示解锁动画
