# GetStar 快速开始指南

**项目**: 儿童行为激励管理微信小程序
**版本**: v1.1 (带用户登录功能)
**日期**: 2026-03-01

---

## 🚀 5分钟快速部署

### 前提条件
- ✅ 微信小程序账号
- ✅ CloudBase账号(腾讯云)
- ✅ 微信开发者工具
- ✅ Node.js v18+

---

## 📝 步骤1: 获取AppID (2分钟)

1. 访问 [微信公众平台](https://mp.weixin.qq.com/)
2. 注册/登录小程序账号
3. 在"开发管理" → "开发设置"中获取 **AppID**

---

## 🔧 步骤2: 配置项目 (1分钟)

修改 `project.config.json`:

```json
{
  "appid": "wxXXXXXXXXXXXXXXXX", // 替换为你的AppID
  "projectname": "GetStar"
}
```

---

## ☁️ 步骤3: 配置CloudBase (2分钟)

### 3.1 创建CloudBase环境

1. 访问 [CloudBase控制台](https://console.cloud.tencent.com/tcb)
2. 点击"新建环境"
3. 环境名称: `getstar-prod`
4. 计费模式: 选择按量付费(新用户有免费额度)
5. 点击"新建"

### 3.2 复制环境ID
- 环境ID格式如: `cloudbase01-xxxxxxxx`

### 3.3 配置数据库安全规则（已自动配置）
本项目的数据库安全规则已自动配置，确保：
- 用户只能访问自己的数据
- 所有数据通过 `_openid` 隔离
- 习惯模板数据所有人可读

详细配置请参考：`docs/微信用户登录功能实现文档.md`

创建成功后,复制环境ID(如: `getstar-prod-xxxxx`)

### 3.3 修改app.js

```javascript
wx.cloud.init({
  env: 'getstar-prod-xxxxx', // 替换为你的环境ID
  traceUser: true,
});
```

---

## 🗄 步骤4: 创建数据库 (3分钟)

在CloudBase控制台 → 数据库,创建以下集合:

| 集合名 | 说明 | 权限 |
|---------|------|--------|
| families | 家庭表 | 所有用户可读,创建者可写 |
| children | 孩子表 | 所有用户可读,创建者可写 |
| habits | 习惯表 | 所有用户可读,创建者可写 |
| check_ins | 打卡记录表 | 所有用户可读,创建者可写 |
| habit_templates | 习惯模板表 | 所有用户可读 |
| daily_records | 每日记录表 | 所有用户可读,创建者可写 |

## ☁️ 步骤6: 部署云函数 (2分钟)

1. 打开微信开发者工具
2. 导入项目(选择项目根目录)
3. 右键 `cloudfunctions/login` → "上传并部署:云端安装依赖"
4. 等待部署完成
5. 右键 `cloudfunctions/checkIn` → "上传并部署:云端安装依赖"
6. 等待部署完成

## ❓ 常见问题

### Q: 编译时报错 "cloud.init is not a function"
**A**: 检查基础库版本,在 `project.config.json` 中设置:
```json
{
  "libVersion": "2.19.4"
}
```

### Q: 云函数调用失败
**A**: 
1. 确认云函数已部署
2. 检查环境ID是否正确
3. 查看CloudBase云函数日志

### Q: 数据库查询失败
**A**: 
1. 确认数据库集合已创建
2. 检查集合权限设置
3. 查看数据库权限配置

### Q: 照片上传失败
**A**: 
1. 确认云存储已开通
2. 检查云存储权限
3. 查看网络连接
