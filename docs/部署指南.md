## 🚀 部署步骤

### 第一步: 配置小程序AppID

1. 登录 [微信公众平台](https://mp.weixin.qq.com/)
2. 注册/登录小程序账号
3. 获取小程序 AppID
4. 修改 `project.config.json`:

```json
{
  "appid": "wxXXXXXXXXXXXXXXXX", // 你的小程序AppID
  "projectname": "GetStar"
}
```

### 第二步: 配置CloudBase环境

1. 登录 [CloudBase控制台](https://console.cloud.tencent.com/tcb)
2. 创建新环境
3. 复制环境ID(如: `getstar-env-xxxxx`)
4. 修改 `app.js`:

```javascript
wx.cloud.init({
  env: 'getstar-env-xxxxx', // 你的环境ID
  traceUser: true,
});
```

### 第三步: 创建数据库集合

在CloudBase控制台的数据库页面,创建以下集合:

#### 1. families (家庭表)
- 权限: 所有用户可读,仅创建者可写

#### 2. children (孩子表)
- 权限: 所有用户可读,仅创建者可写
- 索引: `familyId`, `createTime`

#### 3. habits (习惯表)
- 权限: 所有用户可读,仅创建者可写
- 索引: `childId`, `familyId`, `order`

#### 4. check_ins (打卡记录表)
- 权限: 所有用户可读,仅创建者可写
- 索引: `childId`, `habitId`, `date`, `createTime`

#### 5. habit_templates (习惯模板表)
- 权限: 所有用户可读,仅管理员可写

#### 6. daily_records (每日记录表)
- 权限: 所有用户可读,仅创建者可写
- 索引: `childId`, `date`

### 第四步: 导入习惯模板数据

### 第六步: 部署云函数

#### 使用微信开发者工具部署

1. 打开微信开发者工具
2. 导入项目(选择项目根目录)
3. 右键 `cloudfunctions/login` 文件夹
4. 选择"上传并部署:云端安装依赖"
5. 等待部署完成
6. 对 `cloudfunctions/checkIn` 重复步骤3-5

#### 验证云函数部署

在CloudBase控制台的云函数页面,检查:
- [ ] `login` 函数已部署
- [ ] `checkIn` 函数已部署
- [ ] 函数状态为"正常"

#### 配置云函数权限

在CloudBase控制台为云函数配置权限:
- `login`: 允许云调用
- `checkIn`: 允许云调用

### 第七步: 配置云存储

1. 在CloudBase控制台开通云存储
2. 设置权限:

## 🔧 常见问题

### Q1: CloudBase初始化失败

**错误信息**: `cloud.init is not a function`

**解决方案**:
1. 检查基础库版本 >= 2.2.3
2. 在 `project.config.json` 中设置:
```json
{
  "libVersion": "2.19.4"
}
```

### Q2: 云函数调用失败

**错误信息**: `云函数执行失败: errCode: -1`

**解决方案**:
1. 确认云函数已部署
2. 检查云函数权限配置
3. 查看云函数日志定位错误

### Q3: 数据库权限错误

**错误信息**: `Permission denied`

**解决方案**:
1. 检查数据库集合权限设置
2. 确保 `_openid` 字段正确写入

### Q4: 照片上传失败

**错误信息**: `uploadFile:fail`

**解决方案**:
1. 检查云存储是否已开通
2. 确认云存储权限配置
3. 检查网络连接

### Q5: TabBar图标不显示

**解决方案**:
1. 确认图标文件路径正确
2. 图标大小限制: 40KB
3. 图标尺寸: 81px * 81px

## 📊 性能优化建议

### 数据库优化
1. 为常用查询字段添加索引
2. 使用 `limit()` 限制查询数量
3. 避免深层嵌套查询

### 云函数优化
1. 使用缓存减少数据库查询
2. 优化异步操作
3. 减少返回数据量

### 前端优化
1. 使用 `wx.nextTick()` 优化渲染
2. 图片使用懒加载
3. 避免频繁 setData

---

## 🔒 安全建议

1. **数据库权限**: 严格设置读写权限,避免数据泄露
2. **云存储权限**: 限制上传文件类型和大小
3. **云函数权限**: 仅开放必要的权限
4. **敏感数据**: 避免在前端存储敏感信息
