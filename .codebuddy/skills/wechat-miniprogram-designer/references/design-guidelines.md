# 微信小程序设计规范参考

## 一、界面设计基础

### 屏幕尺寸与单位
- 设计稿基准：750px 宽度（iPhone 6/7/8 标准，375pt × 2倍）
- 单位使用 `rpx`（responsive pixel），750rpx = 100vw
- 图标和图片需提供 @2x / @3x 切图

### 导航栏
- 默认导航栏高度：44px（88rpx）
- 状态栏高度：iOS 约 20px，有刘海屏为 44px
- 使用 `wx.getSystemInfo` 获取真实状态栏高度
- 自定义导航栏时注意 `navigationStyle: custom`

### 标签栏 (TabBar)
- 高度：50px（100rpx）
- 图标尺寸：81×81rpx（未选中），自定义可用 81×81rpx
- 最多支持 5 个 tab，最少 2 个
- 文字大小：10px

## 二、色彩规范

### 微信品牌色
- 主题绿：`#07C160`
- 深绿（悬停/按下）：`#06AD56`
- 链接蓝：`#576B95`

### 通用灰阶
| 用途 | 色值 |
|------|------|
| 主文本 | `#191919` / `#000000` |
| 次文本 | `#666666` |
| 辅助文本 | `#999999` |
| 提示/禁用 | `#CCCCCC` |
| 分割线 | `#EDEDED` |
| 背景 | `#F5F5F5` |
| 卡片背景 | `#FFFFFF` |

### 功能色
- 成功：`#07C160`
- 警告：`#FFC300`
- 危险/错误：`#FA5151`
- 信息蓝：`#10AEFF`

## 三、字体规范

### 字号层级
| 级别 | 大小 | 用途 |
|------|------|------|
| 大标题 | 20px (40rpx) | 页面主标题 |
| 标题 | 17px (34rpx) | 导航栏标题、卡片标题 |
| 正文 | 15px (30rpx) | 主要阅读内容 |
| 辅助文字 | 13px (26rpx) | 描述、说明 |
| 小字 | 11px (22rpx) | 标签、角标、时间戳 |

### 行高建议
- 标题：1.4 倍字号
- 正文：1.6~1.8 倍字号

### 字重
- 加粗（bold/600）：标题、重要数值
- 正常（regular/400）：正文
- 细体（light/300）：辅助描述

---

## 四、间距与圆角

### 间距体系（8pt 网格）
- 基础单位：8rpx
- 常用间距：8 / 16 / 24 / 32 / 40 / 48rpx
- 页面水平边距：32rpx（16px）
- 卡片内边距：24~32rpx

### 圆角规范
| 元素 | 圆角 |
|------|------|
| 按钮 | 8rpx 或圆形（border-radius: 50rpx） |
| 卡片 | 16rpx |
| 弹窗 | 24rpx |
| 头像 | 50%（圆形） |
| 标签 | 4~8rpx 或胶囊形 |

---

## 五、组件交互规范

### 按钮
- 主按钮高度：88rpx（44px），全宽或固定宽度
- 次按钮：与主按钮同高，描边样式
- 文字按钮：无背景，使用链接色
- 危险按钮：红色背景 `#FA5151`
- 禁用状态：透明度 40% 或使用禁用色 `#CCCCCC`

### 列表项
- 单行列表高度：96rpx（48px）
- 多行列表最小高度：96rpx，内容自适应
- 左侧图标：48×48rpx 到 80×80rpx
- 右侧箭头：使用 `>` 或 chevron 图标，颜色 `#CCCCCC`

### 输入框
- 高度：88rpx（44px）
- 边框：1rpx solid `#EDEDED`，focus 时 `#07C160`
- placeholder 颜色：`#CCCCCC`

### 弹窗 / Modal
- 宽度：680rpx（90%屏宽）
- 遮罩：黑色 50% 透明度
- 出现动画：fade + scale from 0.8

---

## 六、图标规范

### 尺寸
- 导航栏图标：44×44rpx
- 列表图标：48×48rpx
- TabBar 图标：81×81rpx
- 操作区图标：64×64rpx

### 格式
- 矢量图标推荐使用 SVG 或 iconfont
- 位图图标提供 @2x 和 @3x
- 单色图标优先使用 font-icon

---

## 七、安全区域

### iPhone X 及以上（有底部 Home Indicator）
- 底部安全区：34px（68rpx）
- 使用 CSS env: `padding-bottom: env(safe-area-inset-bottom)`
- TabBar 场景由框架自动处理

### 自定义导航栏安全区
```js
wx.getSystemInfo({
  success(res) {
    const statusBarHeight = res.statusBarHeight; // 状态栏高度
    const navBarHeight = 44; // 导航栏内容区
    const totalHeight = statusBarHeight + navBarHeight;
  }
})
```

---

## 八、无障碍设计

- 可点击区域最小 44×44px
- 颜色对比度：正文文字与背景对比度 ≥ 4.5:1
- 图片元素提供 aria-label 或 alt 描述
- 交互状态（hover/active/disabled）需有视觉反馈
