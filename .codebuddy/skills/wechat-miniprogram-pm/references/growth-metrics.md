## 一、核心数据指标 {#核心指标}

### 用户规模指标
| 指标 | 定义 | 健康参考值 |
|------|------|----------|
| **DAU** | 日活跃用户数 | 持续增长 |
| **MAU** | 月活跃用户数 | DAU/MAU > 20% 较好 |
| **新增用户** | 首次使用用户数 | 参考行业 |
| **注册用户** | 已授权/登录用户 | 注册率 > 60% |

### 留存指标
| 指标 | 定义 | 健康参考值 |
|------|------|----------|
| **次日留存** | 第2天仍活跃 / 第1天新用户 | 电商 > 20%，工具 > 30% |
| **7日留存** | 第8天仍活跃 / 新用户 | > 10% |
| **30日留存** | 第31天仍活跃 / 新用户 | > 5% |
| **月留存** | 下月活跃 / 本月活跃 | > 40% |

### 活跃度指标
| 指标 | 定义 | 说明 |
|------|------|------|
| **人均使用时长** | 总时长 / 活跃用户数 | 内容类 > 5min |
| **人均访问页面数** | 总 PV / 活跃用户数 | > 3 页为较好 |
| **访问频次** | 月均打开次数/用户 | > 4次/月 |

### 业务转化指标
| 指标 | 计算公式 | 应用场景 |
|------|---------|---------|
| **注册率** | 注册用户 / 访客数 | 通用 |
| **功能渗透率** | 使用某功能 UV / 总 UV | 功能评估 |
| **核心转化率** | 目标动作 / 进入漏斗 UV | 电商/服务 |
| **付费转化率** | 付费用户 / 活跃用户 | 商业化 |
| **ARPU** | 总收入 / 活跃用户数 | 商业化 |

### 传播指标
| 指标 | 定义 | 说明 |
|------|------|------|
| **K 因子** | 每用户带来的新用户数 | > 1 代表病毒式增长 |
| **分享率** | 分享用户 / 活跃用户 | 内容类 > 5% |
| **分享转化率** | 分享带来的新用户 / 分享次数 | > 20% 较好 |

## 二、增长策略框架 {#增长策略}

### AARRR 漏斗模型

```
Acquisition（获客）
  ↓ 渠道：搜索、扫码、分享、广告
Activation（激活）
  ↓ 首次体验到核心价值（"Aha Moment"）
Retention（留存）
  ↓ 用户持续回来使用
Revenue（收入）
  ↓ 付费转化、商业变现
Referral（传播）
  ↓ 用户主动分享、带来新用户
```

### 微信小程序特有增长路径

**冷启动阶段（0→1万用户）**
- 私域导流：公众号、社群、企业微信
- KOC/KOL 合作，精准圈层渗透
- 线下场景：二维码、NFC、门店触达
- 目标：验证核心功能，跑通基本流程

**成长阶段（1万→100万用户）**
- 朋友圈广告（定向投放）
- 微信搜索 SEO 优化
- 附近的小程序（LBS）
- 任务裂变 / 邀请有礼
- 目标：找到可复制的增长引擎

**规模化阶段（100万+）**
- 品牌广告 + 效果广告协同
- 开放平台合作
- 内容生态建设
- 目标：降低边际获客成本，提升 LTV

### 北极星指标（NSM）选择原则
- 只选 **1个** 最能代表产品核心价值的指标
- 必须与业务目标直接相关
- 可以被团队行动影响

| 产品类型 | 推荐北极星指标 |
|---------|--------------|
| 电商 | 周下单用户数 |
| 内容 | 人均阅读时长 |
| 工具 | 日活完成核心任务的用户数 |
| 社区 | 日发内容用户数 |
| 服务预约 | 周预约完成量 |

## 三、用户分层运营 {#用户分层}

### RFM 分层模型
- **R（Recency）**：最近一次使用距今天数
- **F（Frequency）**：使用频次
- **M（Monetary）**：消费金额（或关键行为次数）

| 用户类型 | R | F | M | 运营策略 |
|---------|---|---|---|---------|
| 核心用户（高价值） | 高 | 高 | 高 | VIP 专属权益，防流失 |
| 潜力用户 | 高 | 中 | 中 | 激励提频，转化付费 |
| 沉睡用户 | 低 | 高 | 高 | 召回推送，唤醒活跃 |
| 流失用户 | 低 | 低 | 低 | 低成本召回，放弃高成本挽留 |
| 新用户 | 高 | 低 | 低 | 新手引导，帮助到达 Aha Moment |

### 用户生命周期管理

```
新用户 → 成长用户 → 成熟用户 → 衰退用户 → 流失用户
  ↑___________________(召回)___________________________↓
```

| 阶段 | 判断标准 | 运营重点 |
|------|---------|---------|
| 新用户（0-7天） | 注册 < 7天 | 新手引导、完善资料、体验核心功能 |
| 成长用户（7-30天） | 活跃但付费低 | 提升使用深度，引导付费 |
| 成熟用户（30天以上） | 高频活跃 | 维持活跃，挖掘更多需求 |
| 衰退用户 | 活跃度下降 | 提前预警，推送有价值内容 |
| 流失用户（30天未活跃） | 长期不使用 | 召回活动，重新激活 |

## 四、微信生态流量来源 {#流量来源}

### 自然流量
| 来源 | 特点 | 优化方向 |
|------|------|---------|
| 微信搜索 | 成本低，精准度高 | 优化小程序名称、简介、关键词 |
| 附近的小程序 | 适合本地服务 | 保持活跃，提升评分 |
| 公众号关联 | 私域价值高 | 公众号菜单、文章内嵌入口 |
| 朋友圈/聊天分享 | 口碑传播 | 优化分享卡片，提升内容吸引力 |
| 历史访问 | 回访用户 | 提升首次体验，触发记忆点 |

### 微信搜索优化（小程序 SEO）
- **名称优化**：包含核心业务关键词（如「XX外卖」「XX预约」）
- **简介优化**：前30字最重要，自然包含场景关键词
- **功能页标题**：页面 `navigationBarTitleText` 设置相关关键词
- **内容索引**：开启「微信索引」功能，让内容页被搜索收录

### 付费流量
| 渠道 | 适用场景 | 参考 CPM/CPC |
|------|---------|------------|
| 朋友圈广告 | 品牌曝光 + 转化 | CPM ¥20-60 |
| 公众号广告 | 精准内容场景触达 | CPC ¥1-5 |
| 微信搜索广告 | 高意向用户 | CPC ¥2-10 |
| 小程序广告联盟 | 流量互换 | CPM ¥5-20 |

### 分享裂变设计要点
1. **分享动机**：利他（「帮朋友砍价」）> 自利（「我得奖励」）
2. **分享卡片**：标题 < 27 字，图片 5:4 比例，信息清晰
3. **落地承接**：分享进来的用户有专属欢迎页/奖励
4. **分享时机**：在用户完成关键成功动作后引导分享（下单成功、获得成就）
5. **合规要求**：禁止「分享给 N 人得奖励」等强制性裂变

## 五、数据分析方法 {#数据分析}

### A/B 测试规范

**实验设计步骤**：
1. 明确假设：「我认为 [改变X] 会导致 [指标Y 提升 Z%]」
2. 确定分流比例：通常对照组 50%，实验组 50%
3. 计算所需样本量：使用统计显著性计算器（显著性 p < 0.05）
4. 设定实验周期：至少 7 天（覆盖完整周期效应）
5. 避免：同时运行多个相互影响的实验

**小程序 A/B 分流方案**：
- 按 `openid` 尾号分组（稳定，不会来回切换）
- 使用微信 AB 实验平台 或 自建分流系统

### 漏斗分析模板
```
[步骤1] → [步骤2] → [步骤3] → [目标]
  100%        75%       40%      15%
              ↑流失25%  ↑流失35% ↑流失25%
              分析原因  分析原因  分析原因
```

**流失节点排查问题清单**：
- 是否有明显的 UI 障碍？
- 加载是否过慢？
- 引导文案是否清晰？
- 该步骤是否必要？（能否合并/简化）

### 用户行为路径分析
- 识别最常见路径（Happy Path）
- 识别意外路径（用户绕开了你设计的流程）
- 识别死胡同（用户到达后直接离开的页面）

### 同期群分析（Cohort Analysis）
按注册周/月追踪用户留存曲线，判断：
- 留存曲线是否趋于平稳（有核心用户群）
- 不同期群留存是否在提升（产品在改进）
- 平稳留存率是多少（产品 PMF 程度）
