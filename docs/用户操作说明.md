# 用户操作说明

## 1. 孩子管理
### 1.1 添加孩子
1. 在首页点击“添加孩子”按钮，或在“我的家庭”区域点击“+”按钮。
2. 输入孩子姓名。
3. 选择头像：
   - **卡通头像**：从预设的男孩/女孩卡通形象中选择。
   - **自定义头像**：点击相机图标（📷），从手机相册选择照片或直接拍摄。支持裁剪功能，可调整头像显示区域。
4. 点击“保存”按钮完成添加。

### 1.2 切换孩子
- **首页**：如果添加了多个孩子，首页顶部会出现孩子列表，点击头像即可切换当前显示的孩子。
- **管理页**：在“孩子管理”页面，点击非当前孩子的“切换”按钮（→）即可切换。

### 1.3 每日打卡
1. 在首页“今日习惯”列表中，找到已完成的习惯。
2. 点击右侧的打卡按钮（+）。
3. 打卡成功后，按钮变为绿色对勾（✓），并获得相应的星星和金币奖励。

### 1.4 查看奖励
1. 点击首页的“兑换礼物”按钮，进入兑换中心。
2. 浏览可兑换的礼物列表，支持按分类筛选。

### 1.5 兑换礼物
1. 点击想要兑换的礼物卡片。
2. 确认兑换信息（消耗金币数）。
3. 点击“确认兑换”，系统将扣除相应金币并记录兑换历史。

## 2. 习惯养成

### 2.1 添加习惯
1. 进入“习惯管理”页面。
2. 点击“添加习惯”按钮。
3. 可以选择系统推荐的习惯模板，或自定义习惯名称、图标和奖励星星数。

## 3. 打卡记录

### 3.1 查看打卡记录
1. 进入打卡记录页面
2. 选择日期可以查看那天的打卡记录
3. 当天打卡的记录可以在24小时内撤销

## 4. 我的

### 4.1 编辑孩子信息
1. 点击孩子头像，进入“孩子管理”页面。
2. 在弹出的窗口中，可以修改孩子姓名或重新选择头像。
3. 点击“保存”按钮更新信息。


## 4. 排行榜
1. 点击首页底部的“排行榜”图标。
2. 查看孩子在家庭中的星星排名。
3. 支持按“本周”、“本月”或“总榜”查看排名。
