# GetStar 快速开始指南

**项目**: 儿童行为激励管理微信小程序
**版本**: v1.2
**日期**: 2026-03-07

---

## 🚀 5分钟快速部署

### 前提条件
- ✅ 微信小程序账号
- ✅ CloudBase账号(腾讯云)
- ✅ 微信开发者工具
- ✅ Node.js v18+

---

## 📝 步骤1: 获取AppID

1. 访问 [微信公众平台](https://mp.weixin.qq.com/)
2. 注册/登录小程序账号
3. 在"开发管理" → "开发设置"中获取 **AppID**

---

## 🔧 步骤2: 配置项目

修改 `project.config.json`:

```json
{
  "appid": "wxXXXXXXXXXXXXXXXX", // 替换为你的AppID
  "projectname": "GetStar"
}
```

---

## ☁️ 步骤3: 配置CloudBase

1. 访问 [CloudBase控制台](https://console.cloud.tencent.com/tcb)
2. 创建新环境（建议按量付费）
3. 复制环境ID(如: `getstar-env-xxxxx`)
4. 修改 `app.js`:

```javascript
wx.cloud.init({
  env: 'getstar-env-xxxxx', // 替换为你的环境ID
  traceUser: true,
});
```

---

## ☁️ 步骤4: 部署云函数

1. 打开微信开发者工具
2. 导入项目
3. 右键 `cloudfunctions` 目录 → "更多设置" → "上传并部署：云端安装依赖"
4. 等待所有云函数部署完成

---

## 🗄 步骤5: 初始化数据库

1. 在微信开发者工具中，打开 "云开发" 控制台
2. 选择 "云函数" -> "initDatabase"
3. 点击 "云端测试"
4. 等待执行完成，这将自动创建所有数据库集合并导入初始数据

---

## 🚀 步骤6: 运行项目

1. 在微信开发者工具点击"编译"
2. 检查控制台是否有错误
3. 点击"预览"生成二维码
4. 微信扫码真机测试
