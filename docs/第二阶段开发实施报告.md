### 2. 云函数开发

#### 已部署云函数 (4个)
| 函数名 | 功能 | 状态 |
|--------|------|------|
| getLeaderboard | 获取积分排行榜 | ✅ 已部署 |
| undoCheckIn | 撤销打卡 | ✅ 已部署 |
| getRewardList | 获取礼物列表 | ✅ 已部署 |
| exchangeReward | 兑换礼物 | ✅ 已部署 |

#### 云函数功能说明

**getLeaderboard**
- 支持按周/月/总榜查询
- 计算每个孩子的总积分、打卡次数、连续打卡天数
- 自动排序和排名
- 前三名特殊标识

**undoCheckIn**
- 24小时内可撤销打卡
- 自动扣减积分和金币
- 记录金币变动历史
- 权限验证

**getRewardList**
- 支持按分类筛选礼物
- 按金币消耗升序排列
- 分类：美食、娱乐、活动、学习、玩具

**exchangeReward**
- 验证金币余额
- 扣减金币并创建兑换记录
- 记录金币变动
- 权限验证
